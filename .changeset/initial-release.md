Initial public release of the Lens Journal toolkit.

- `@lens-journal/core`: EXIF-aware photo pipeline, journal-entry schema, config loader, and validation shared by the CLI and theme.
- `@lens-journal/theme`: Tailwind preset, base styles, and framework-agnostic view-model helpers.
- `lens-cli`: the `lens` command for importing photos and managing entries.
- `create-lens-journal`: `npm create lens-journal` scaffolder.
